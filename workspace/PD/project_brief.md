## 1) 一句话定义
- 从“扰动前后转录组表型指纹”出发，**反向推断**最可能的扰动/驱动靶标基因（单靶标或多靶标集合），并提供网络级可解释路径与多场景验证闭环的深度学习框架。【来源：项目背景；PD项目主文档.pdf】

## 2) 背景与动机
- 虚拟细胞主流是“从因到果”的正向扰动响应预测；真实任务常是“从果到因”的逆向推断：给定表型变化，追溯驱动扰动源头。【来源：PD项目主文档.pdf】
- 传统差异表达/富集分析更擅长描述“是什么变了”，但难回答“是什么导致了这种变化”；需要机制一致性的反向推断框架。【来源：PD项目主文档.pdf】

## 3) 项目目标与成功指标（当前从主文档抽取，后续会结合日志/结果细化）
- 目标：突破现有方法在（1）全基因组搜索受限，（2）未见靶标零样本泛化弱，（3）可解释性缺失，（4）验证不足 的瓶颈。【来源：PD项目主文档.pdf】
- 核心评测划分：未见细胞系 / 未见基线 / 未见靶标（真零样本）。【来源：PD项目主文档.pdf】
- 主要指标：nDCG、Recall@K、靶标平均排名；化合物多靶点增加“部分准确率”。【来源：PD项目主文档.pdf】

## 4) 技术方案（数据 / 模型 / 训练 / 评估 / 可解释性）
### 4.1 数据与评测基准（Task 1）
- 数据：基于 LINCS Level 3 构建 bulk 扰动前后严格配对数据集：
  - CRISPR 单靶标扰动；化合物多靶标扰动。
  - 处理：剔除无效样本、按实验条件配对、对照组 MODZ 聚合、统一基因ID与标准化、标签术语标准化。
  【来源：PD项目主文档.pdf】
- 三类挑战划分：留一细胞系；严格按基线切分；靶标独立（测试靶标训练未出现）。【来源：PD项目主文档.pdf】

### 4.2 模型架构（Task 2）
- 总体：主分支端到端预测 + 对齐分支机制约束（对比学习对齐“表型偏移”与“网络传播指纹”）。【来源：PD项目主文档.pdf】
- 主分支：BulkFormer（并比较 scGPT/scFoundation 等）编码 pre/post 表达谱 → 差分得到表型偏移 z=upost-upre → MLP 解码头输出全基因组打分并排序。【来源：PD项目主文档.pdf】
- 轻量微调：冻结主干 + 插入 LoRA 等可训练模块；pre/post 两侧共享参数保证同空间。【来源：PD项目主文档.pdf】
- 对齐分支：真实靶标 multi-hot 门控注入基因语义向量 → PPI 上用 LCAT 扩散得到传播指纹 → 投影到对齐空间，与表型偏移做 InfoNCE 对比对齐。【来源：PD项目主文档.pdf】
- 多模态基因语义：ESM2 蛋白序列 + Pfam 结构域 + GO 功能（Onto2Vec 等）融合；用 EMA 平滑更新以稳定对齐训练。【来源：PD项目主文档.pdf】

### 4.3 可解释性（Task 2-解释输出）
- 样本级稀疏解释子图：Attention×Gradient 提取关键边/节点形成传播路径子图。【来源：PD项目主文档.pdf】
- 反事实必要性证据：掩码关键边/节点后对齐分数显著下降作为证据；随机掩码作对照。【来源：PD项目主文档.pdf】
- 跨样本共识：聚合同靶标/同药物多样本解释子图形成稳健机制线索。【来源：PD项目主文档.pdf】

## 5) 当前进展（待从工作日志1-7与结果 sources 完整抽取；这里先写已知骨架）
- 已完成：简化版“双分支对齐”预实验（主分支 BulkFormer 差分表型偏移；对齐分支基于 ESM2 语义先验 + InfoNCE）。【来源：PD项目主文档.pdf】
- 结果：在未见细胞系/未见基线任务上显著优于 PDGrapher 与 FoldChange；未见靶标（零样本）仍有提升空间，指向需要引入 Pfam+GO 与 PPI 扩散等机制级先验。【来源：PD项目主文档.pdf】

## 6) 关键风险与应对（来自主文档）
- 核心风险：未见靶标零样本泛化提升不确定；对策是多模态语义 + PPI 扩散 + 对比对齐，并保留更丰富先验/更强图模型/蒸馏/元学习等备选策略。【来源：PD项目主文档.pdf】
- 技术风险：双分支训练不稳定、对齐失效、多模态融合不理想；对策：EMA、门控融合、系统消融与模块化定位问题。【来源：PD项目主文档.pdf】
