# 单细胞扰动响应预测算法基准测试简报

## 执行摘要

本报告基于《Nature Methods》（2025年发表）的一项深度基准测试研究，旨在评估用于预测单细胞扰动响应的计算方法。该研究针对 27 种算法（包括新兴的基础模型、深度学习模型、图神经网络模型及基准模型）进行了全面评估，涵盖了 29 个数据集和 6 项评价指标。

核心结论指出，目前不存在“一劳永逸”的方法。算法在“细胞背景泛化”和“扰动泛化”两个主要场景下的表现差异显著。研究特别强调，虽然基础模型在充足的微调数据下表现优异，但在数据稀疏或背景差异巨大的情况下，简单的线性模型或基准模型往往更具竞争力。本报告旨在为研究人员在具体实验条件下选择最合适的预测工具提供实操指南。

---

## 核心主题详细分析

### 1. 评估框架与场景定义
研究将单细胞扰动预测任务分为两个关键维度：
*   **细胞背景泛化场景 (Cellular Context Generalization)：** 在已知扰动的情况下，预测该扰动在未观察到的细胞背景（如不同细胞系、患者或物种）中的效应。
*   **扰动泛化场景 (Perturbation Generalization)：** 在特定细胞背景下，预测从未见过的扰动（遗传或化学扰动）所产生的效应。

评估通过 **准确性**（人口平均指标如 MSE、PCC-delta、E-distance；人口分布指标如 Wasserstein、KL 散度、Common-DEGs）、**鲁棒性**（对噪声和数据稀疏性的敏感度）以及 **可扩展性**（运行时间和内存消耗）进行。

### 2. 细胞背景泛化：间际异质性的挑战
*   **表现优异者：** trVAE、CellOT 和 inVAE 在单条件数据集上表现最佳。若涉及化学扰动的剂量信息，**scVIDR** 是最优选择。
*   **关键障碍：** 所有方法的预测性能高度依赖于测试背景与训练背景之间的相似性（间际异质性）。当预测与训练集差异巨大的新细胞背景时，现有方法表现均较差，甚至不如简单的基准模型。
*   **协变量的影响：** 时间点和剂量等协变量对细胞响应有实质性影响。尽管部分模型（如 biolord、scDisInFact）尝试引入这些信息，但在预测精度上仍有较大提升空间。

### 3. 扰动泛化：数据规模与模型复杂度的权衡
*   **遗传扰动：** 
    *   **单基因扰动：** 当训练数据较小时，**GenePert**（基于 LLM 嵌入的线性模型）表现最优；当微调数据充足时，**CPA** 和 **scGPT** 等深度学习及基础模型展现出更强的泛化能力。
    *   **组合扰动：** **linearModel** 和 **scouter** 表现最佳。对于加性交互作用，简单模型表现稳健；而对于增强、上位性或冗余等复杂交互，CPA 等深度模型具有一定优势。
*   **化学扰动：** 
    *   单药预测中 **chemCPA** 凭借其在 bulk 数据上的预训练优势表现突出。
    *   组合药物预测仍是难点，基础模型目前普遍不如简单的 **baseReg** 基准模型。

### 4. 基础模型与大语言模型 (LLM) 嵌入
研究发现，利用 ChatGPT 衍生文本描述作为嵌入的算法（如 GenePert 和 scouter）在某些场景下优于利用转录组驱动嵌入的模型。这表明 LLM 嵌入可能捕捉到了从纯基因表达数据中难以提取的高级功能和背景知识。

## 重要引言与背景

> “我们的研究表明，不存在一种‘通用型’方法能适用于所有数据集。目前单细胞研究中迫切需要细胞背景嵌入方法，以增强扰动效应预测的泛化能力。”

**背景说明：** 此引言强调了当前预测模型在跨细胞类型泛化方面的局限性，指出算法设计往往忽略了不同背景下扰动响应的特异性。

> “令人惊讶的是，依赖于未对齐自然语言嵌入的模型表现优于利用转录组信息嵌入的模型……这种优势可能源于 LLM 衍生嵌入中蕴含的丰富生物学先验知识。”

**背景说明：** 研究人员通过消融实验证实，性能提升并非源于模型架构的复杂性，而是源于嵌入层所包含的功能性信息。

## 工具选择指南与行动建议

根据基准测试结果，针对不同的实验需求和数据特征，建议采取以下策略：

### 扰动响应预测工具建议表

| 预测场景 | 数据特征 | 推荐工具/模型 | 核心优势 |
| :--- | :--- | :--- | :--- |
| **细胞背景泛化** | 单一条件 (Single-condition) | **trVAE / inVAE** | 人口平均指标（MSE）表现最优 |
| | 包含剂量信息 | **scVIDR** | 能够利用剂量协变量信息 |
| | 关注差异表达基因 (DEGs) | **scPRAM** | 在 Common-DEGs 分布指标上更准确 |
| **遗传扰动泛化** | 数据量小 (Small-scale) | **GenePert** | 基于 LLM 嵌入，利用生物学先验 |
| | 数据量大 (Large-scale) | **CPA / scGPT** | 随微调数据增加，泛化性能显著提升 |
| | 基因组合扰动 | **linearModel / scouter** | 捕捉交互效应的能力较强 |
| **化学扰动泛化** | 单药处理 | **chemCPA** | 预训练策略有效利用了 bulk 数据 |
| | 药物组合处理 | **baseReg (基准模型)** | 现有高级模型在组合预测上仍有欠缺 |

### 行动见解
1.  **数据稀疏性管理：** 所有算法对数据稀疏性的敏感度高于噪声。在数据质量较差时（高稀疏度），应谨慎解读深度模型的结果，可能需要参考线性基准模型的表现。
2.  **引入外部先验：** 开发者应考虑将基因调控网络、通路信息或细胞系嵌入引入模型设计，以克服当前模型泛化能力不足的局限。
3.  **损失函数优化：** 现有模型多侧重于优化人口平均指标。未来研究应尝试将人口分布指标（如 Wasserstein 距离）纳入损失函数，以更好地捕捉扰动后的细胞异质性。
4.  **计算资源考量：** **scFoundation** 虽为基础模型，但内存消耗巨大且运行时间长；**CellOT** 不支持 GPU 加速，处理大规模数据时效率较低。在资源有限时应优先选择 trVAE 或基础线性模型。
